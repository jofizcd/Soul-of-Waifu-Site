Установка Soul of Waifu

Установка Soul of Waifu - простой процесс, который займет всего несколько минут. Следуйте инструкциям ниже для успешной установки приложения.

Скачивание программы

Для получения последней версии Soul of Waifu вы можете воспользоваться одним из двух способов:

Через сайт

Перейдите на официальный сайт проекта и нажмите кнопку "Скачать для Windows" для получения последней стабильной версии программы.

Скачать Soul of Waifu

Через GitHub

Вы можете скачать программу с GitHub:

  1. Перейдите на репозиторий проекта на GitHub
  2. Откройте вкладку "Releases"
  3. Выберите последнюю версию программы
  4. Скачайте архив с программой

Распаковка архива

После скачивания архива с программой:

  1. Найдите скачанный архив
  2. Перенесите архив в папку, где вы хотите установить программу
  3. Распакуйте архив с помощью любого архиватора (например, WinRAR или 7-Zip)

Запуск установщика

После распаковки архива:

  1. Откройте папку с распакованной программой
  2. Найдите файл installer.bat
  3. Запустите его не от имени администратора
  4. Выберите нужную версию PyTorch - CUDA (видеокарты с поддержкой CUDA) или CPU
  5. Дождитесь завершения процесса установки (это может занять несколько минут)
Важно: Во время установки программа может выводить ошибки, связанные с rvc-python, но в этом нет ничего критичного. Убедитесь, что у вас стабильное интернет-соединение.

Запуск программы

После завершения установки:

  1. В папке с программой найдите файл start.bat и запустите его
  2. Дождитесь открытия Soul of Waifu

Системные требования

  • ОС: Windows 10/11
  • ОЗУ: Минимум 8 ГБ (рекомендуется 16 ГБ)
  • Видеокарта: Современная видеокарта с поддержкой CUDA (для локальных моделей)
  • Место на диске: Минимум 10 ГБ свободного пространства
  • Дополнительно: Доступ к интернету для первоначальной настройки
